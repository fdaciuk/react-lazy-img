# React Lazy Image

## Usage

```js
import LazyImg from 'react-lazy-img'

// ...

<LazyImg src='path/to/img.jpg' onImageLoad={() => console.log('image loaded!')} />
```

The CSS class `-loaded` is automatically added when image is ready.
